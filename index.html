このツールでできること

調べたい銘柄の証券コードを入力するだけで、「今の株価は割安か、割高か」を2つの角度から自動判定します。

計算方法はマネックス証券「銘柄スカウター」と同じ方式を採用。Yahoo Finance からリアルタイムで財務データを取得して計算するため、インストール不要でHTMLファイルをブラウザで開くだけで使えます（日本株・米国株に対応）。

インターネット接続が必要です。このページ（Claude.aiのプレビュー）では外部APIに接続できないため、一部銘柄のサンプルデータが表示されます。ローカルにHTMLファイルを保存してブラウザで開くとリアルタイムデータで動作します。
操作手順
  1. 証券コードを入力する
    日本株は4桁の数字（例：7203 = トヨタ）、米国株はティッカー（例：AAPL = Apple）を入力します。「.T」は自動付加されるので不要です。
  2. 「計算する」を押す
    最初はこれだけでOKです。PBR基準・PER基準の理論株価が自動で表示されます。EPS成長率は暫定で8%がセットされた状態で計算されます。
  3. 成長率を推定して精度を上げる
    計算後に「成長率を推定する」ボタンが有効になります。クリックすると過去実績や アナリスト予想から3種類の推定値が表示されます。気に入った値をクリックするとスライダーに自動セットされます。その後、再度「計算する」を押すと精度が上がります。
  4. （任意）PER・EPSを手動で入力する
    自分の業績見通しがある場合は、スイッチをONにして手動入力できます。自動取得より信頼できる数値がある場合に使います。
PBR基準カードの見方

PBR基準は「その銘柄が過去75日間、どのPBR水準で売買されてきたか」を統計的に分析して理論株価を出す手法です。PBR（株価純資産倍率）とは、株価が1株あたり純資産の何倍かを示す数字で、値が低いほど資産に対して株が安い状態です。

計算のイメージ
理論株価 = BPS（純資産/株）x 過去75日のPBR平均
上値目途 = BPS x （PBR平均 + 2シグマ）　← 上昇余地の上限目安
下値目途 = BPS x （PBR平均 - 2シグマ）　← 下落リスクの下限目安
表示項目読み方
大きい数字（理論株価） カード上部に大きく表示される金額が「PBR基準の理論株価」です。隣にある 緑の+% は現在株価より割安、赤の-% は割高を意味します。
上値目途（+2シグマ） 過去の値動きから見た「上がりすぎ」の目安。この価格に近づくと短期的に天井感が出やすい水準です。
下値目途（-2シグマ） 過去の値動きから見た「下がりすぎ」の目安。この価格に近づくと短期的に底値感が出やすい水準です。
BPS（1株純資産） 計算の土台となる数値。会社が解散したときに1株あたり戻ってくる純資産の理論値です。
現在PBR / 75日平均 / シグマ 今の株価のPBRが、過去75日の平均からどれだけ離れているかを確認できます。現在PBRが平均より大きく高ければ割高圏、大きく低ければ割安圏のシグナルです。
PER基準カードの見方

PER基準は「将来の利益（EPS）を予測して、それに株価の倍率（PER）をかけて理論株価を出す」手法です。PBR基準が過去の値動きから算出するのに対し、PER基準は将来の業績見通しを反映できるのが特徴です。

計算のイメージ
将来EPS = 今のEPS x （1 + 年間成長率） の N乗　← 複利で成長させる
理論株価 = 将来EPS x 想定PER

たとえば「EPS 500円、成長率10%、3年後、PER 15倍」なら、将来EPS = 500 x 1.1^3 = 665円、理論株価 = 665 x 15 = 約9,980円になります。

表示項目読み方
大きい数字（理論株価） カード上部に表示される「将来の理論株価」です。現在株価との比較が % で表示されます。
基準EPS 成長計算のスタート地点となる1株利益。自動で来期予想EPSを取得します。自分の見立てがある場合は手動入力スイッチでON。
成長率 x 年数 スライダーで設定した「年間成長率」と「予測年数」の組み合わせです。成長率が高いほど・年数が長いほど理論株価が高くなります。
予測EPS（N年後） 基準EPSに成長率を複利で積み上げた将来の利益予測。この数値が計算のキーになります。
想定PER 自動では実績PERと予想PERの平均を使います。業種や成長性に応じて手動で変えると精度が上がります（成長株なら高め、安定株なら低め）。
EPS成長率の推定機能について

「EPS成長率なんて分からない」という方のために、計算実行後に「成長率を推定する」ボタンが表示されます。財務データを自動分析して3種類の推定値を提示します。

推定方法内容と特徴
過去3年 EPS CAGR 直近3年間の実績EPSの年平均成長率。「最近の実力値」を反映しやすいが、好不況の影響を受けやすい。景気敏感株などは数値がぶれることがある。
過去5年 EPS CAGR 直近5年間の実績EPSの年平均成長率。3年よりも長い期間を見るため、短期ノイズが平滑化されてより安定した傾向が見える。中長期投資の参考に向く。
アナリスト予想 証券アナリストが予想する将来の成長率の中央値。主に米国株で取得できることが多い。日本株は取得できない場合もある。将来の見通しを重視したい場合に参考にする。
推定値はあくまで「過去の実績」や「アナリストの予想」に基づく参考値です。将来の成長を保証するものではありません。3つの数値を見比べながら、自分の見通しに合った値を選ぶか、スライダーで微調整して使うのがお勧めです。
結果の読み方まとめ
表示意味
+XX%（緑） 理論株価 ＞ 現在株価。現在の株価は計算上の適正水準より安い「割安」状態の可能性があります。
-XX%（赤） 理論株価 ＜ 現在株価。現在の株価は計算上の適正水準より高い「割高」状態の可能性があります。
PBRとPERで結果が違う よくあることです。PBRは過去の値動き基準、PERは将来の利益基準なので、見ている軸が異なります。両方の結果を参考にしながら総合的に判断するのが基本です。
重要：理論株価は「この計算式に当てはめるとこうなる」という参考値です。実際の株価には需給・金利・センチメントなど無数の要因が影響するため、理論株価どおりに動く保証はありません。投資判断はご自身の責任で行ってください。
よくある質問
質問回答
証券コードがわからない 日本株はYahoo!ファイナンスやマネックス証券のサイトで銘柄名を検索すると4桁コードを確認できます。米国株はティッカー記号（AAPL、MSFTなど）を入力してください。
PBR基準が表示されない BPS（1株純資産）データが取得できなかった場合に発生します。特に米国株では取得できないことがあります。その場合はPER基準のみをご参照ください。
PER基準が表示されない EPSデータが自動取得できなかった場合です。「基準EPSを手動入力」スイッチをONにして、決算短信などから1株利益を手入力してください。
「成長率を推定する」が使えない オフライン（デモ）モードでは利用できません。ローカルのHTMLファイルをブラウザで開き、「計算する」を実行した後に有効になります。
成長率の推定値が3つとも「取得不可」 EPSの過去データがYahoo Finance上に存在しない銘柄は推定できません。スライダーを手動で調整するか、業績資料を参考に手動入力してください。
「DEMO」と表示されている リアルタイムAPIに接続できない環境（プレビュー等）で、内蔵のサンプルデータを使って表示しています。参考値として確認できますが実際の株価・財務データとは異なります。
